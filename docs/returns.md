# Polityka zwrotów

## 30-dniowa gwarancja zwrotu

Nie zadowolony? Zwróć produkt w ciągu 30 dni!

## Jak zwrócić produkt?

1. **Skontaktuj się**: [returns@susstyle.com](mailto:returns@susstyle.com)
2. Otrzymasz etykietę zwrotną
3. Zapakuj produkt (nieużywany, metki)
4. Wyślij za darmo
5. Zwrot pieniędzy w 5-7 dni

## Warunki

- Produkt **nieużywany**
- Oryginalne opakowanie i metki
- Paragony/faktury
- Zwrot w ciągu **30 dni** od otrzymania

## Co nie podlega zwrotowi?

- Produkty na wymianie
- Produkty uszkodzone przez użytkownika
- Produkty bez metek lub opakownia
